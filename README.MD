## Descripción del proyecto  
El proyecto consiste en el desarrollo de una aplicación de consola en lenguaje C++ diseñada para facilitar la administración y control de inventario en un entorno comercial básico. El sistema permite al usuario gestionar una base de datos en memoria de hasta 100 productos, ofreciendo herramientas para el registro, consulta, modificación y análisis de datos.

La aplicación se estructura bajo los principios de la programación estructurada y modular, implementando un menú interactivo que centraliza el flujo de operaciones. Entre sus funcionalidades técnicas destacan la implementación de tres algoritmos de ordenamiento (Burbuja, Selección e Inserción) para organizar productos por precio, algoritmos de búsqueda lineal para la localización de artículos y la generación de reportes estadísticos que calculan el valor total del inventario y detectan productos con mayor stock.


El sistema permite **registrar productos**, **mostrarlos**, **ordenarlos**, **actualizarlos**, **buscar por nombre** y **generar un reporte estadístico final**.

## Habilidades y conceptos aplicados 
- Implementación de algoritmos de ordenamiento 
- Construcción de un menú interactivo usando `switch`  
- Uso de **structs** para modelar productos  
- Manipulación de **arreglos unidimensionales**  
- Modularidad mediante **funciones**  
- Validación de datos ingresados por el usuario  
- Uso de ciclos `for`, `while` y `do–while`
